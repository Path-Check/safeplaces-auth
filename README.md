# SafePlaces Information Security

The modular information security service for the SafePlaces backend.

```shell script
# Install using NPM
npm install @pathcheck/safeplaces-auth@^2.1.0

# Install using Yarn
yarn add @pathcheck/safeplaces-auth@^2.1.0
```
